游戏内容：
通过不断挑战敌人、获得奇遇，提升自己的攻击和防御，自己变强的同时，敌人也在变强，尽量在最少的轮数中获得超高的攻击力、防御力。

游戏机制：
从起点到终点，在未达到一定的目的前不断轮回；
以获得的奇遇，在当前轮次中不会再获得，下一轮重置；
以挑战的敌人，在当前轮次中不能再次挑战，下一轮将以成长后的属性出现；
